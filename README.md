# ionic-performance
Performance tests for Ionic Framework:

- Virtual Scroll
- Infinite Scroll
- List Filtering
- Skeleton Screen
- Animation

Usage

1

Install [Node.js]


2
```sh
$ git clone https://github.com/alessandronunes/ionic-performance.git
$ cd ionic-performance
$ npm install
$ ionic serve
```

[node.js]: <http://nodejs.org>
